Setup Guide

Complete these steps once to connect Meta Lead Ads

1
Create a Meta App
Go to developers.facebook.com/apps, create a new app of type Business, and add the Facebook Login for Business and Webhooks products.
2
Configure the Webhook in Meta
In your app's Webhooks settings, subscribe to the leadgen field on the Page object.
Set Callback URL to the Webhook URL shown below. Set Verify Token to the value you generate in the Configuration section below.
3
Generate a Long-Lived Page Access Token
In the Graph API Explorer, select your app and your Facebook Page, then generate a token with pages_manage_metadata, pages_read_engagement, and leads_retrieval permissions.
4
Subscribe your Page to the App
Using the Graph API Explorer, call:
POST /{page-id}/subscribed_apps?subscribed_fields=leadgen&access_token={page-access-token}
5
Fill in credentials below and enable
Enter the App Secret, Page Access Token, and Verify Token in the Configuration section. Save, then toggle the integration to Enabled.
